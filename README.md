# multihash

An implementation of [multihash]([https://multiformats.io/multihash/), as seen in [IPFS](https://ipfs.io/).

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  multihash:
    github: teknomunk/cr-multihash
```

## Usage

```crystal
require "multihash"
```
